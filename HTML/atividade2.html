A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language,
que significa Linguagem de Marcação de Hipertexto) é uma linguagem
de marcação utilizada na construção de páginas na Web. Documentos HTML
podem ser interpretados por navegadores.

Alguns dos recursos da linguagem HTML vistos até este momento:

Recursos Tag Associadas Descrição
Links <a> Criar Links entre paginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar listas de itens nao ordenados
Imagens <img> Inserir imagens
